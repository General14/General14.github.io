Many people think coding websites is very hard and can only be made by experienced coders. This is not true. Websites are coded with HTML, CSS and JS*. This website was coded with these things. My guide to doing this will, hopefully, help you to make your own website.

Before you continue it is important you find an IDE (Integrated development environment), this is where you can write your code. Currently I am using a desktop IDE called Microsoft Visual Studio Code. There are lots them but if you are just starting coding, I would suggest replit.com in repl they give you all the files, and there are packages to help add things. I will show instructions in repl, I would choose this one.

In repl you should first make an account. Choose a wise username as this will be part of the URL (Uniform resource locator), the link to type in the adress bar.
Then make a new HTML, CSS and JS file. On repl, you have to press the small blue plus sign , or click on this link. If you do this you will need to press the fork button to edit your own copy.

*Some websites are made with a Python module called Flask
