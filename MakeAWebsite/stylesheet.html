CSS

CSS stands for Cascading Style Sheet. This is what makes up how a website looks. Here is an example: body { background-color: blue } This will make the background colour blue. In your repl you will see a link tag. This is why you can see the style. Make sure when you use CSS you reference the elemnt you want to change (see above). There are 3 types of CSS, inline, internal or external.

Inline

This is when you add style to a specific element.

Internal

This is when you use the style tags to style only one page.

External

This is when you link a css file to all the html pages. We will be using External CSS.

Instructions

Now you can follow these steps to add some style to your website.

Start by opening the style.css file and typing body { background-color: blue; } You can change the colour to anything, this is an example.
Now you can find a font using Google Fonts. Once you have chosen, this can take some time, go into your body tag in your CSS file and type this: font-family: (name your font);. Make sure to press select this style. You will see in Google Fonts a link starting with @import copy this and paste it into the top of the CSS file.
Then use the color to change the colour of the text. Then your website should look very different. Remember 1 thing: there should be a semi-colon after averything in CSS. For more colors go to this website.
Completed Code
style.css
@import url('https://fonts.googleapis.com/css2?family=Roboto&display=swap');
body {
background-color: #f0f0f0;
color: darkslategray;
font-family: Roboto;
}
