HTML

HTML stands for Hypertext Markup Language. This is what makes up all web pages. HTML code is very easy, this is what it looks like: <p> . In HTML make sure that with every open tag (see above) you need a closing tag: </p>. There are a few exceptions including, the <link> tag. In your repl you should have lots of elements including a title, body and head.

Title

This is what your website will say on the tab.

Body

This is where the main part of your website will go

If you want to learn more elements go to Elements.

Instructions

Now you can follow these steps to add some content to your website.

Change the <title> to whatever that you want.
Now add a <h1> and add whatever you want the title to anything. You can change it to h2 or h3 or even h6.
Then add a <p> and make the paragraph with anything.
Now make a list: put a <ul> and then a <li> tag make the list with anything.
Make sure to have end tags or the whole thing will be one paragraph.
