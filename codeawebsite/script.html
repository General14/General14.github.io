JS

JS stands for JavaScript. This is what makes your website be interactive with buttons and other things. Here is an example: alert('Hello World!') This will make the background colour blue. In your repl you will see a link tag. This is why you can see the style. Make sure when you use CSS you reference the elemnt you want to change (see above). There are 3 types of CSS, inline, internal or external.

Instructions
On your script.js file start with typing alert('Hello'). Press run and you should see an alert with the text you put in speech marks.
On your HTML file, type:
<p>Hello
<span class="mytext" id="text"></span>
<button onClick = "myText()">My Button</button>
</p>
On your JS file, type:
function myText() {
my_elem = document.getElementById("text");
my_elem.innerHTML = 'You pressed a button!';
my_elem.style.color = 'gray';
}
