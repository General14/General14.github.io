Most people think coding websites is extremely hard and only for very good coders. This is not true. Websites are coded with HTML, CSS and JS. This website was coded with these things. My guide to doing this will, hopefully, help you to make your own website.

Before you continue it is important you find an IDE (Integrated development environment), this is where you can write your code. Currently I am using a desktop IDE called Microsoft Visual Studio Code. There are lots but if you are just starting coding, I would suggest repl.it in repl they give you all the files, there is a community and some examples to help you get going. There is also GitHub, The problem with this is GitHub is very complicated. So I would stick with repl. I will show instructions in repl, so it is probably the best choice.

In repl you should first make an account. Choose a wise username as this will be part of the URL (Uniform resource locator).
Then make a new HTML, CSS and JS file. On repl, you have to press the small blue plus sign.
After that you should follow my instructions.
